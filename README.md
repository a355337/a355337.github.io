# 🌈 魔法數字樂園 | 兒童雙語數學思維啟蒙遊戲

歡迎來到 **魔法數字樂園 (Magic Number Land)**！這是一個專為幼兒、學齡前與低年級兒童量身打造的 **雙語趣味互動數數與數學思維啟蒙網頁遊戲**。

本專案完全基於前端網頁技術建構，擁有色彩繽紛、可愛活潑的視覺風格，並配備完整的中英雙語語音發音與即時獎勵回饋機制，旨在讓孩子在「玩中學、學中玩」，自然而然建立敏銳的數感與英語聽力基礎。

## 🚀 三大精彩遊戲模式

本遊戲內含三款精心設計的核心挑戰：

### 1. 🍓 水果大混合 (不規則數數挑戰)
*   **玩法：** 畫面中會散落多種繽紛的水果。孩子需要根據中/英文語音提示（例如：「請數一數有幾個草莓？」/ "Please count how many strawberries are there?"），找出並點擊對應的水果來進行計數。
*   **特色：** 點擊對的水果時，畫面會出現清晰的數字標籤，並伴隨可愛的音效，協助孩子練習一對一的點數對應。

### 2. 🎈 數字氣球碰！ (快速反應與數字識別)
*   **玩法：** 帶有數字的彩色氣球會不斷從畫面下方飄浮上升。孩子需要聆聽語音指令，在時限內精準找出並點擊帶有指定數字的氣球，將其「砰」一聲戳破！
*   **特色：** 氣球移動速度會隨著難度動態調整，完美訓練孩子的眼手協調度、視覺辨識與聽力反應能力。

### 3. ⚖️ 糖果誰最多？ (數感與大小對比)
*   **玩法：** 畫面左右兩側將分別出現裝有不同數量可愛甜點（如棒棒糖、甜甜圈）的卡片。孩子需要快速比一比，點選數量比較多的那一邊！
*   **特色：** 不需要逐個精細數數，藉由直觀的視覺面積與分布，培養快速判斷多寡的「直覺數感」與邏輯思考能力。

## ✨ 核心專案亮點

*   **🗣️ 中英雙語語音導覽：** 遊戲全程皆提供流暢清晰的中/英雙語發音提示。任何時候都可以點擊「🔊 重播聲音」按鈕，讓孩子反覆聆聽，輕鬆進行雙語學習。
*   **⭐ 智慧星級獎勵機制：** 答對即可獲得閃亮的金色星星！答題速度越快，還能獲得額外的連勝倍率加乘。當孩子順利通關時，會有華麗的紙花特效與讚美音效，最大程度激發成就感與遊玩興趣。
*   **📱 極致流暢的觸控優化：** 完美支援手機、iPad、安卓平板等多種行動裝置。使用大字體與大點擊範圍的圓角按鈕，避免誤觸，操作感極佳。
*   **🔒 純前端安全無干擾：** 
    *   零廣告、零後端資料收集、無不當內容。
    *   純 HTML5、CSS (Tailwind CDN)、Web Speech API。
    *   不需下載安裝任何 App，點開即玩，給孩子最純淨安全的學習環境。
